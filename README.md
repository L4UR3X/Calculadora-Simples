# Calculadora Simples

Projeto de **calculadora simples em JavaScript, HTML e CSS** que realiza operações básicas (adição, subtração, multiplicação e divisão) diretamente no navegador.

---

## Sobre

Este repositório contém uma interface web de calculadora, com:
- Interface em **HTML**
- Estilo com **CSS**
- Lógica com **JavaScript**

Ideal para praticar programação front-end básica com manipulação do DOM.

---

Arquivos principais:
index.html
style.css
script.js
